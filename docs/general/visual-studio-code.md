If you are using Visual Studio Code here's some recommended extensions for it.

## Recommended Extensions

* [EditorConfig](https://open-vsx.org/extension/EditorConfig/EditorConfig)
* [Better Comments](https://open-vsx.org/extension/aaron-bond/better-comments)
* [Color Highlight](https://open-vsx.org/extension/naumovs/color-highlight)
* [Encode Decode](https://open-vsx.org/extension/mitchdenny/ecdc)
* [GitLens](https://open-vsx.org/extension/eamodio/gitlens)
* [Markdown All in One](https://open-vsx.org/extension/yzhang/markdown-all-in-one)
* [Markdown Preview Mermaid Support](https://open-vsx.org/extension/bierner/markdown-mermaid)
* [ShellCheck](https://open-vsx.org/extension/timonwong/shellcheck)
* [YAML](https://open-vsx.org/extension/redhat/vscode-yaml)
